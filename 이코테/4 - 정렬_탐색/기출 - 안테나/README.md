# 기출 - 안테나

## 느낀점
* 

## notes
* 동일위치에 여러집 존재 가능.
* 모든 집까지의 거리의 총합이 최소인 곳에 세우기.
* 집이 위치한 곳에만 안테나 설치 가능.
* 여러 값이 도출될 경우 가장 작은 값 출력

## input
```
4 // 집 4개
5 1 7 9 // 각각의 위치
```

## output
```
5 // 안테나는 5에 세워져야함.
```

## strategy
* 정렬한담에 가운데 있는 곳에 세우면 될까?
  * 그러면 될듯.
* 짝수개 있을때 왼쪽가운데나 오른쪽가운데나 똑같을 것 같다.
