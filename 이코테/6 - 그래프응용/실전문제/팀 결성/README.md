```
1 1 7 // f(1) == f(7) ?

1 7 1
0 3 7
0 4 2
0 1 1
1 1 1
```

```
NO // 1번과 7번은 같은 팀이 아니다.
NO // 7번과 1번은 같은 팀인가?
YES // 1번과 1번은 같은 팀인가?
```
