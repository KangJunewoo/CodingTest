# 경로탐색 - 다익스트라

## 느낀점
* 다익스트라 한 절반정도 외우고 있었으니
* 이 참에 싹 외워보자 ㅋㅋ
* DFS/BFS vs 다익스트라
  * 가중치 무, 가중치 유
  * visit_t, dist_t
  * 스택큐에 v가 들어감, 힙에 (dist, v)로 기존 그래프와 거꾸로들어감.
  * 공통적으로 인접노드를 이용함.
