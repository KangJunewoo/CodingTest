## input

```
5 : 0시0분0초~5시59분59초까지 중 3이 하나라도 포함되는 경우의 수는
```

```
11475 : 11475개이다.
```

## strategy
* 하나라도 포함되는 경우의 수 == 전체 - 전혀 포함되지 않는 경우의 수
  * 전체 : (N+1) * 60 * 60
  * 3없이 : (x) * 45 * 45 (3 포함된 수가 15가지임.)
    * N이 3 이상일때, x는 N
    * N이 0, 1, 2일때, x는 (N+1)
