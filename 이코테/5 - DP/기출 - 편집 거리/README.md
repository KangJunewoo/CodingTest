## input
```

```

## output
```

```

## strategy
* 느낌은 dp이다.
* 결국엔 모두 변환은 가능하다. 문자열차이 + (변화된수)니까 변화된수를 최소로 줄여야 하는 것.
* 아님 그냥 그리디로 해도 되지 않나?
  * 전문자열 후문자열 카운터 돌리고 len(B) - 공통부분 합
* 근데 이렇게 했더니 아무리 많이 겹쳐있어도 순서가 정반대면 위치 문제때문에 결국 삭제와 삽입이 이뤄져야 한다.
* 답보자~
